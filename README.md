# poly2pascal
Convert POLYGON bounding boxes from `.csv` files to `.xml` annotation files in the Pascal VOC format for computer vision projects.

## Installation
Install `poly2pascal` with pip
```bash
  pip install poly2pascal
```

## Requirements
* pencv-python==4.6.0.66
* geopandas==0.12.1
